Falls du nicht weiter weißt, kannst du einen der unten stehenden Tipps zur Rate ziehen:

Du musst beide Schriftgrößenangaben anpassen, um die Aufgabe zu lösen. Überleg genau, welche Werte passen könnten.

Passende Werte für eine Standardschriftgröße sind 100%, 1rem, 1em, large oder medium. Ersetze die Pixelangaben für jede eingeblendete Schriftgröße auf einen dieser Werte.
